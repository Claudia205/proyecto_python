# proyecto_python
Proyecto enfocado en el uso de python, pandas, análisis de datos, IA e interés por la programación

Este Bootcamp a tenido como objetivo impulsar a los estudiantes en el aprendizaje, trabajando en modalidad virtual hemos podido mejorar nuestros conocimientos en analisis de datos, python y el uso de la inteligencia artificial.

El Salvador 2025 LAMARR-ESPAÑA

Este proyecto utiliza Python, Pandas y IA para analizar datos de diferentes practicas, entre ellas la creación de un programa enfocado en videojuegos donde se otorgaban variables como por ejemplo de vidas del jugador.

## Tecnologías
- Python
- Pandas
- Colab
